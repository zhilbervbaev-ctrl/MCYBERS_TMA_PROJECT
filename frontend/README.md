# GDPR Auditor - Frontend Dashboard

A React-based web interface for displaying GDPR and ePrivacy audit results.

## Requirements
*   Node.js 18+
*   Backend API running on `http://localhost:3000`

## Installation

```bash
npm install
```

## Running the Dashboard

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the application.
